Языки разметки

Исследуем HTML, YAML, YML и XML - основные языки для структурирования данных в вебе и не только

🌐
HTML

HTML (HyperText Markup Language) - стандартный язык разметки для создания веб-страниц и веб-приложений.

<!-- Пример HTML документа -->
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Заголовок страницы</title>
</head>
<body>
  <h1>Привет, мир!</h1>
  <p>Это пример HTML документа.</p>
</body>
</html>

Использование: Веб-разработка, создание веб-страниц и приложений

📄
YAML

YAML (YAML Ain't Markup Language) - человеко-читаемый язык сериализации данных, использующий отступы для структуры.

# Пример YAML документа
person:
  name: Иван Иванов
  age: 30
  hobbies:
    - чтение
    - программирование
    - путешествия
  address:
    city: Москва
    country: Россия

Использование: Конфигурационные файлы, CI/CD, Docker Compose, Ansible

⚙️
YML

YML - это то же самое, что и YAML. Расширение .yml часто используется вместо .yaml из-за ограничений в старых системах.

# Пример .yml файла конфигурации
database:
  host: localhost
  port: 5432
  name: myapp
server:
  port: 8080
  environment: development
features:
  enabled:
    - authentication
    - logging
    - caching

Использование: Конфигурационные файлы приложений, GitHub Actions, Travis CI

📋
XML

XML (eXtensible Markup Language) - язык разметки для хранения и передачи структурированных данных.

<!-- Пример XML документа -->
<?xml version="1.0" encoding="UTF-8"?>
<catalog>
  <book id="1">
    <title>Война и мир</title>
    <author>Лев Толстой</author>
    <year>1869</year>
  </book>
  <book id="2">
    <title>Преступление и наказание</title>
    <author>Фёдор Достоевский</author>
    <year>1866</year>
  </book>
</catalog>

Использование: Веб-сервисы (SOAP), конфигурации, документы (Office Open XML), RSS

Сравнение языков разметки

Язык Тип Читаемость Сложность Основное применение
HTML Язык разметки Средняя Низкая Веб-страницы
YAML Язык сериализации Высокая Низкая Конфигурации, данные
XML Язык разметки Средняя Высокая Структурированные данные
